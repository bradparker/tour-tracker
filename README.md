# Tour Tracker

Plot tourers along a route using their distance.

[See it in action!](https://bradparker.github.io/tour-tracker/)

### The Bits

* React
* Leaflet js
* Redux (optional)

To view the examples:

```
$ npm i
$ npm run dev
```

Open your browser at `localhost:8080`
